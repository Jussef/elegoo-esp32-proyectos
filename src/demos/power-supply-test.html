Descripción
Cómo se usa Electrónica · Alimentación

Aprende a usar el Power Supply Module para alimentar tu ESP-32 con una batería de 9V y dejarlo funcionar sin cable USB. El módulo recibe los 9V de la batería y los convierte en 5V estables que entrega a los rieles del protoboard. Desde ahí llevas el + al pin VIN del ESP-32 y el − al GND. El pequeño jumper de plástico de cada riel es el que selecciona el voltaje de salida: lo pones en 5V.

Cómo funciona el módulo

El módulo se monta a horcajadas sobre el protoboard y alimenta sus dos rieles laterales (el + y el − de cada lado). Cada lado tiene su propio jumper con 3 posiciones:

Posición del jumperSalida de ese riel
3.3VEl riel entrega 3.3V
5VEl riel entrega 5V  ←  el que usamos
OFF (jumper quitado o en el centro)Ese riel queda apagado
Cada lado se configura por separado. Puedes tener, por ejemplo, un riel en 5V para el ESP-32 y el otro en 3.3V para un sensor, al mismo tiempo.

Cuida la polaridad: el − va a GND y el + a VIN. Invertirlos puede dañar la placa. Y no conectes el riel de 5V al pin 3V3 del ESP-32 — ese pin es salida de 3.3V, no entrada.
Código para confirmar que enciende

No hace falta ningún código especial para alimentarlo, pero este parpadeo del LED interno te confirma de un vistazo que el ESP-32 arrancó con la batería. Súbelo por USB y luego pásalo a la batería.

C++ / Arduino IDE (ESP-32) 
// Latido de "estoy vivo": confirma que el ESP-32 corre con la bateria
#define LED 2   // LED integrado de la mayoria de placas ESP-32 DevKit

void setup() {
  pinMode(LED, OUTPUT);
}

void loop() {
  digitalWrite(LED, HIGH);   // encendido
  delay(200);
  digitalWrite(LED, LOW);    // apagado
  delay(800);
}
Cómo usar
1
Monta el Power Supply Module sobre el protoboard, encajando sus patas en los rieles + y − de los lados.
2
Pon el jumper del lado que vas a usar en la posición 5V.
3
(Opcional) Sube el código del latido al ESP-32 por USB primero, para tenerlo listo.
4
Cablea: riel − → GND del ESP-32, y riel + → VIN.
5
Conecta la batería de 9V a la entrada del módulo y enciéndelo con su interruptor. El LED debe empezar a parpadear: ¡el ESP-32 ya corre con la batería!
No alimentes por USB y por VIN a la vez. Para programar usa el USB; para correr con batería, desconecta el USB. Tener las dos fuentes conectadas al mismo tiempo puede causar problemas.
Sobre la duración: una batería de 9V tiene poca capacidad. Sirve perfecto para pruebas y proyectos cortos, pero con WiFi encendido se agota rápido. Para algo permanente, una fuente por el jack o un power bank por USB rinden mucho más.
