Descripción
Demo de prueba OLED · I2C

La prueba más sencilla para tu pantalla OLED: muestra "Hola Mundo!" en grande. Sirve para confirmar que la pantalla está bien conectada por I2C, que tiene la dirección correcta (casi siempre 0x3C) y que las librerías están bien instaladas. Si ves el texto, ¡tu OLED está lista para cualquier proyecto!

Conexiones
ComponentePin ESP-32Notas
OLED — SDAGPIO 21Bus I2C (datos)
OLED — SCLGPIO 22Bus I2C (reloj)
OLED — VCC3.3V—
OLED — GNDGND—
¿Pantalla en negro? La mayoría de estas OLED usan la dirección 0x3C, pero algunas usan 0x3D. Si no aparece nada, cambia ese valor en oled.begin(...).
Código Arduino
C++ / Arduino IDE (ESP-32) 
#include <Wire.h>
#include <Adafruit_GFX.h>
#include <Adafruit_SSD1306.h>

#define ANCHO 128
#define ALTO  64
Adafruit_SSD1306 oled(ANCHO, ALTO, &Wire, -1);

void setup() {
  Serial.begin(115200);
  Wire.begin(21, 22);   // SDA=21, SCL=22

  // Si no arranca, prueba la direccion 0x3D
  if (!oled.begin(SSD1306_SWITCHCAPVCC, 0x3C)) {
    Serial.println("No se encontro la OLED. Revisa cables o prueba 0x3D.");
    while (true);   // se queda aqui: la pantalla no respondio
  }

  oled.clearDisplay();
  oled.setTextColor(SSD1306_WHITE);

  oled.setTextSize(2);
  oled.setCursor(20, 12);
  oled.println("Hola");
  oled.setCursor(8, 36);
  oled.println("Mundo!");

  oled.display();   // sin esta linea, no se ve nada
  Serial.println("OLED OK: texto enviado.");
}

void loop() {
  // Nada que repetir: el texto queda fijo en pantalla
}
Cómo usar
1
Instala desde el Gestor de Librerías Adafruit SSD1306 y su dependencia Adafruit GFX (el IDE te ofrece instalarlas juntas).
2
Conecta la OLED: SDA→21, SCL→22, VCC→3.3V y GND→GND.
3
Sube el código. En unos segundos deberías ver "Hola Mundo!" en la pantalla.
4
Si queda en negro, abre el Monitor Serie (115200): si dice que no encontró la OLED, cambia 0x3C por 0x3D y vuelve a subir.
Detalle clave: con esta librería nada aparece hasta que llamas a oled.display(). Todo lo que dibujas (texto, líneas) se guarda primero en memoria y solo se "vuelca" a la pantalla con esa línea.
