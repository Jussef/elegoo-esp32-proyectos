Descripción
Demo de prueba Motor · PWM

La prueba más sencilla para tu motor/ventilador de 3-6V: lo enciende y lo apaga, y luego varía su velocidad usando PWM. Como un GPIO del ESP-32 no puede mover un motor directamente (le falta corriente), lo controlamos con un transistor PN2222 que hace de interruptor, más un diodo de protección. El Monitor Serie te va diciendo a qué velocidad está girando en cada momento.

Conexiones
ConexiónVa aNotas
ESP-32 — GPIO 25Base PN2222A través de una resistencia de 1 kΩ
Emisor PN2222GNDGND común con el ESP-32
Colector PN2222Motor (−)El transistor abre/cierra el negativo del motor
Motor (+)5V (VIN)Alimenta el motor desde 5V
Diodo 1N4007en paraleloSobre el motor, banda hacia el + (protección)
Nunca conectes el motor directo a un GPIO. El motor pide más corriente de la que da el pin y lo dañaría. El transistor hace de interruptor y el diodo de retorno (1N4007 en paralelo con el motor, banda hacia el +) absorbe el pico de tensión que genera la bobina al apagarse. Sin el diodo, ese pico puede dañar el transistor o el ESP-32.
Código Arduino
C++ / Arduino IDE (ESP-32) 
#define PIN_MOTOR  25   // GPIO 25 -> (resistencia 1k) -> base del PN2222

// PWM por hardware del ESP-32 (LEDC)
const int CANAL    = 0;      // canal LEDC 0..15
const int FREC     = 1000;   // 1 kHz, suave para un motor pequeno
const int RES_BITS = 8;      // resolucion 8 bits -> valores 0..255

void setup() {
  Serial.begin(115200);
  delay(300);
  ledcSetup(CANAL, FREC, RES_BITS);   // configurar el canal PWM
  ledcAttachPin(PIN_MOTOR, CANAL);    // enlazar el GPIO a ese canal
  Serial.println("Motor listo. Probando velocidades...\n");
}

void setVelocidad(int pct) {          // pct: 0..100 %
  int duty = map(pct, 0, 100, 0, 255);
  ledcWrite(CANAL, duty);
  Serial.printf("Velocidad: %3d %%  (duty %d/255)\n", pct, duty);
}

void loop() {
  // Subir de 0 a 100 % poco a poco
  for (int v = 0; v <= 100; v += 25) {
    setVelocidad(v);
    delay(1500);
  }
  // Bajar de 100 a 0 %
  for (int v = 75; v >= 0; v -= 25) {
    setVelocidad(v);
    delay(1500);
  }
  Serial.println("--- ciclo completo ---");
  delay(1000);
}
Qué verás en el Monitor Serie
Motor listo. Probando velocidades... Velocidad: 0 % (duty 0/255) Velocidad: 25 % (duty 63/255) Velocidad: 50 % (duty 127/255) Velocidad: 75 % (duty 191/255) Velocidad: 100 % (duty 255/255) Velocidad: 75 % (duty 191/255) --- ciclo completo ---
El motor debe arrancar suave e ir acelerando hasta el máximo y luego frenar. Es normal que a 25 % el motor pequeño aún no arranque (necesita un mínimo de empuje); empezará a girar a partir de cierto punto. Si gira y cambia de velocidad con cada paso, ¡funciona!
Cómo usar
1
Monta el transistor: GPIO 25 → resistencia 1kΩ → base, emisor → GND, colector → motor(−).
2
Conecta motor(+) → 5V y coloca el diodo 1N4007 en paralelo con el motor, con la banda hacia el +.
3
No hace falta instalar ninguna librería: el PWM usa ledc, que ya viene con el core del ESP-32.
4
Sube el código, abre el Monitor Serie a 115200 baudios y observa cómo el motor sube y baja de velocidad.
¿El motor no gira? Revisa que el emisor vaya a GND (no el colector), que la base tenga la resistencia de 1 kΩ y que el motor reciba 5V. ¿Gira siempre al máximo o nunca? Comprueba que el GPIO de la base sea el 25 y que el GND del ESP-32 y el de la alimentación del motor estén unidos.
