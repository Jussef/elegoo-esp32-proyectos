Demo de prueba Control · Analógico

La prueba más sencilla para tu Joystick Module: lee la posición de los dos ejes (X e Y) por las entradas analógicas y el botón que se activa al presionar la palanca hacia abajo. Imprime todo en el Monitor Serie y además te dice la dirección en la que estás empujando (arriba, abajo, izquierda, derecha o centro). Sirve para confirmar que el joystick está bien conectado y que los valores cambian al moverlo. No necesita pantalla ni librerías externas.

Conexiones
ComponentePin ESP-32Notas
Joystick — +5V (VCC)3.3VAliméntalo a 3.3V para que los valores no pasen del rango del ADC
Joystick — GNDGND—
Joystick — VRxGPIO 34Eje X (entrada analógica, solo lectura)
Joystick — VRyGPIO 35Eje Y (entrada analógica, solo lectura)
Joystick — SWGPIO 32Botón (se lee con resistencia pull-up interna)
Importante: aliméntalo a 3.3V, no a 5V. El ADC del ESP-32 solo mide hasta 3.3V, así que con 5V los extremos del eje se saturarían. GPIO 34 y 35 son entradas solo de lectura, perfectas para los ejes analógicos.
Código Arduino
C++ / Arduino IDE (ESP-32) 
#define PIN_X   34   // VRx -> eje X (entrada analogica)
#define PIN_Y   35   // VRy -> eje Y (entrada analogica)
#define PIN_SW  32   // SW  -> boton (se presiona la palanca)

void setup() {
  Serial.begin(115200);
  delay(300);
  pinMode(PIN_SW, INPUT_PULLUP);   // el boton conecta a GND al pulsar
  Serial.println("Joystick listo. Muevelo y presiona la palanca...\n");
}

void loop() {
  int x = analogRead(PIN_X);          // 0..4095 (ADC de 12 bits)
  int y = analogRead(PIN_Y);          // 0..4095
  bool pulsado = (digitalRead(PIN_SW) == LOW);  // LOW = presionado

  // Decidir la direccion segun que tan lejos del centro (~2048) esta cada eje
  String dir = "CENTRO";
  if (y > 3000)      dir = "ABAJO";
  else if (y < 1000) dir = "ARRIBA";
  else if (x > 3000) dir = "DERECHA";
  else if (x < 1000) dir = "IZQUIERDA";

  Serial.printf("X:%4d  Y:%4d  Boton:%s  ->  %s\n",
                x, y, pulsado ? "SI" : "no", dir.c_str());

  delay(200);
}
Qué verás en el Monitor Serie
Joystick listo. Muevelo y presiona la palanca... X:2010 Y:2035 Boton:no -> CENTRO X:2008 Y:4090 Boton:no -> ARRIBA X:4095 Y:2041 Boton:no -> DERECHA X: 12 Y:2038 Boton:no -> IZQUIERDA X:2015 Y:2030 Boton:SI -> CENTRO
En reposo los dos ejes marcan cerca de 2048 (el centro). Al empujar la palanca, uno de los valores se va hacia 0 o hacia 4095 y cambia la dirección. Al presionar la palanca hacia abajo, Boton pasa a SI. Si todo eso pasa, ¡tu joystick funciona!
Cómo usar
1
Conecta el joystick: +5V→3.3V, GND→GND, VRx→34, VRy→35 y SW→32.
2
No hace falta instalar ninguna librería: el código lee los ejes con analogRead() y el botón con digitalRead(), todo del core del ESP-32.
3
Sube el código y abre el Monitor Serie a 115200 baudios.
4
Mueve la palanca en las 4 direcciones y presiónala: los valores de X, Y y Boton deben reaccionar al instante.
¿Los ejes salen al revés o no llegan a los extremos? Es normal que cada joystick tenga su centro un poco distinto; ajusta los umbrales (3000 y 1000) a tu gusto. Si un eje no cambia, revisa que VRx y VRy estén en GPIO 34 y 35, y que la alimentación sea 3.3V.
