Descripción
Demo de prueba ESP-32 · LED integrado

Este es el "Hola Mundo" del hardware: hace parpadear el LED que ya viene soldado en la placa ESP-32. Sirve para confirmar de un vistazo que el código se sube correctamente y que tu placa, el cable USB y el IDE están bien configurados, sin necesidad de conectar ningún componente extra. Si el LED parpadea, ¡todo listo para empezar los demás proyectos!

Conexiones
ComponentePin ESP-32Notas
LED integrado en la placa GPIO 2 No necesitas cablear nada: ya está en la placa
¿No parpadea? Algunas placas usan otro pin para el LED integrado (común: 2, a veces 22 o ninguno). Si no ves nada, conecta un LED externo con una resistencia de 220Ω entre GPIO 2 y GND.
Código Arduino
C++ / Arduino IDE (ESP-32) 
#define LED 2   // LED integrado en la mayoria de placas ESP-32

void setup() {
  pinMode(LED, OUTPUT);
  Serial.begin(115200);
  Serial.println("Blink iniciado. Si lees esto, el codigo se subio bien.");
}

void loop() {
  digitalWrite(LED, HIGH);   // enciende
  Serial.println("LED ON");
  delay(500);

  digitalWrite(LED, LOW);    // apaga
  Serial.println("LED OFF");
  delay(500);
}
Cómo usar
1
Conecta la ESP-32 al PC con el cable USB. En el IDE de Arduino selecciona la placa "ESP32 Dev Module" y el puerto COM correcto.
2
Copia el código y pulsa Subir (la flecha →). Si tu placa lo pide, mantén presionado el botón BOOT mientras empieza a subir.
3
Cuando termine, el LED azul de la placa empezará a parpadear cada medio segundo.
4
Abre el Monitor Serie a 115200 baudios para ver los mensajes LED ON / LED OFF. ¡Si ves esto, tu entorno funciona perfecto!
