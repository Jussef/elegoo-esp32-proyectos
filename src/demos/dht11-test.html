Descripción
Demo de prueba Sensor · Digital 1-Wire

La prueba más sencilla para tu sensor DHT11: lee la temperatura (en °C) y la humedad relativa (en %) y lo imprime en el Monitor Serie. Calcula además la sensación térmica a partir de ambos valores. Sirve para confirmar que el sensor está bien conectado y que responde. Usa una sola línea de datos (protocolo 1-Wire propio de Maxim), y solo necesita la librería DHT de Adafruit.

Conexiones
ComponentePin ESP-32Notas
DHT11 — VCC (+)3.3VTambién funciona a 5V; 3.3V es lo más seguro para el ESP-32
DHT11 — DATA (S)GPIO 4Línea de datos digital
DHT11 — GND (−)GND—
Solo 3 cables. El módulo DHT11 del kit (placa azul de 3 pines) ya trae la resistencia de pull-up incorporada, así que no necesitas añadir ninguna. Si usaras el sensor suelto de 4 patas, pon una resistencia de 10 kΩ entre DATA y VCC.
Código Arduino
C++ / Arduino IDE (ESP-32) 
#include <DHT.h>

#define DHT_PIN   4        // DATA del DHT11 -> GPIO 4
#define DHT_TYPE  DHT11    // tipo de sensor

DHT dht(DHT_PIN, DHT_TYPE);

void setup() {
  Serial.begin(115200);
  delay(300);
  dht.begin();
  Serial.println("DHT11 listo. Leyendo cada 2 segundos...\n");
}

void loop() {
  // El DHT11 necesita al menos 1-2 s entre lecturas
  float hum  = dht.readHumidity();        // humedad relativa (%)
  float temp = dht.readTemperature();     // temperatura (grados C)

  // readHumidity/readTemperature devuelven NaN si la lectura falla
  if (isnan(hum) || isnan(temp)) {
    Serial.println("ERROR: no se pudo leer el DHT11. Revisa el cable de DATA.");
    delay(2000);
    return;
  }

  // Sensacion termica (heat index) calculada por la libreria
  float st = dht.computeHeatIndex(temp, hum, false);

  Serial.printf("Temp: %.1f C   Humedad: %.0f %%   Sensacion: %.1f C\n",
                temp, hum, st);

  delay(2000);
}

Prueba a echar tu aliento cerca del sensor: la humedad sube enseguida y la temperatura sube poco a poco. Si sostienes el sensor en la mano, la temperatura también sube. El DHT11 es lento y poco preciso (±2 °C, ±5 % HR), así que no esperes decimales perfectos. ¡Si los valores reaccionan, funciona!
Cómo usar
1
Conecta el DHT11: +→3.3V, S(DATA)→GPIO 4 y −→GND. Solo 3 cables.
2
En el Arduino IDE instala la librería "DHT sensor library" de Adafruit desde el Gestor de librerías (acepta instalar también Adafruit Unified Sensor si lo pide).
3
Sube el código y abre el Monitor Serie a 115200 baudios.
4
Echa tu aliento o calienta el sensor con la mano: Temp y Humedad deben cambiar en pocos segundos.
¿Sale "no se pudo leer el DHT11" o NaN? Revisa que el pin de DATA esté en GPIO 4, que la alimentación esté firme y que no leas más rápido de cada 2 segundos. El DHT11 necesita ese tiempo entre medidas.
