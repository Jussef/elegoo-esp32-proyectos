Descripción
Demo de prueba Sensor · Distancia

La prueba más sencilla para tu sensor ultrasónico HC-SR04: mide la distancia a un objeto enviando un pulso de sonido por TRIG y midiendo cuánto tarda el eco en volver por ECHO. Imprime la distancia en centímetros en el Monitor Serie. Sirve para confirmar que el sensor está bien conectado y que reacciona al acercar o alejar la mano. No necesita librerías externas.

Conexiones
ComponentePin ESP-32Notas
HC-SR04 — VCC5V (VIN)El HC-SR04 necesita 5V para funcionar bien
HC-SR04 — TRIGGPIO 5Salida: dispara el pulso ultrasónico
HC-SR04 — ECHOGPIO 18Entrada: devuelve el eco (¡ver nota!)
HC-SR04 — GNDGND—
¡Atención con ECHO! El pin ECHO entrega 5V y los GPIO del ESP-32 son de 3.3V. Lo ideal es bajar ese voltaje con un divisor de tensión (por ejemplo 1 kΩ y 2 kΩ) entre ECHO y el GPIO. Para una prueba corta suele tolerarse directo, pero el divisor protege tu placa.
Código Arduino
C++ / Arduino IDE (ESP-32) 
#define PIN_TRIG  5    // TRIG -> GPIO 5  (salida)
#define PIN_ECHO  18   // ECHO -> GPIO 18 (entrada, idealmente con divisor)

void setup() {
  Serial.begin(115200);
  delay(300);
  pinMode(PIN_TRIG, OUTPUT);
  pinMode(PIN_ECHO, INPUT);
  digitalWrite(PIN_TRIG, LOW);
  Serial.println("HC-SR04 listo. Midiendo distancia...\n");
}

void loop() {
  // 1) Pulso de disparo de 10 us en TRIG
  digitalWrite(PIN_TRIG, LOW);
  delayMicroseconds(2);
  digitalWrite(PIN_TRIG, HIGH);
  delayMicroseconds(10);
  digitalWrite(PIN_TRIG, LOW);

  // 2) Medir cuanto dura el eco (en microsegundos). Timeout 30 ms (~5 m)
  long dur = pulseIn(PIN_ECHO, HIGH, 30000);

  if (dur == 0) {
    Serial.println("Fuera de rango (sin eco).");
  } else {
    // 3) Distancia = (tiempo * velocidad del sonido) / 2
    //    velocidad ~ 0.0343 cm/us, dividido entre 2 (ida y vuelta)
    float cm = dur * 0.0343 / 2.0;
    Serial.printf("Distancia: %.1f cm\n", cm);
  }

  delay(300);
}

Acerca y aleja la mano frente al sensor: la distancia debe bajar y subir al instante. Si apuntas al vacío o a más de ~4 m, saldrá Fuera de rango. El HC-SR04 mide bien desde unos 2 cm hasta ~4 m. ¡Si los números reaccionan, funciona!
Cómo usar
1
Conecta el HC-SR04: VCC→5V, TRIG→GPIO 5, ECHO→GPIO 18 (mejor con divisor) y GND→GND.
2
No hace falta instalar ninguna librería: el código usa pulseIn(), que ya viene con el core del ESP-32.
3
Sube el código y abre el Monitor Serie a 115200 baudios.
4
Pon la mano o un objeto plano frente al sensor y muévelo: la distancia debe cambiar en tiempo real.
¿Siempre sale "Fuera de rango" o valores absurdos? Revisa que VCC sea 5V (no 3.3V), que TRIG y ECHO no estén intercambiados y que el objeto sea plano (las superficies blandas o en ángulo dispersan el eco). Apunta a una pared para una prueba fiable.
