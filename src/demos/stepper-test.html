Descripción
Demo de prueba ULN2003 · Stepper 28BYJ-48

La prueba más simple del motor paso a paso: gira una vuelta completa en un sentido y luego una vuelta en el otro, sin parar. Sirve para confirmar que el cableado de las 4 bobinas está bien, que el driver ULN2003 recibe corriente y que el motor responde, antes de pasar a los proyectos reales. Si el eje gira de ida y vuelta de forma suave, ¡todo está conectado correctamente!

Motor 28BYJ-48—Conector blanco de 5 pines a la placa ULN2003
Pista visual: los 4 LEDs de la placa ULN2003 (A, B, C, D) deben encenderse en secuencia mientras el motor gira. Si parpadean pero el eje no se mueve, suele ser el conector del motor mal puesto o falta de alimentación de 5V.
Código Arduino
C++ / Arduino IDE (ESP-32) 
#include <Stepper.h>

const int STEPS_REV = 2048;   // 28BYJ-48: pasos por vuelta completa

#define IN1 19
#define IN2 18
#define IN3 5
#define IN4 17

// El orden IN1, IN3, IN2, IN4 da la secuencia correcta del 28BYJ-48 con ULN2003
Stepper motor(STEPS_REV, IN1, IN3, IN2, IN4);

void setup() {
  Serial.begin(115200);
  motor.setSpeed(12);   // velocidad en RPM (10-15 es estable)
  Serial.println("Test stepper iniciado: una vuelta en cada sentido.");
}

void loop() {
  Serial.println("Girando en sentido horario...");
  motor.step(STEPS_REV);     // una vuelta completa (+)
  delay(800);

  Serial.println("Girando en sentido antihorario...");
  motor.step(-STEPS_REV);    // una vuelta completa (-)
  delay(800);
}
Cómo usar
1
La librería Stepper.h ya viene con el IDE de Arduino, no instalas nada.
2
Enchufa el motor a la placa ULN2003 (cable blanco de 5 pines, entra de una sola forma) y cablea IN1–IN4 a los pines indicados.
3
Alimenta el driver con la fuente de 5V externa y une todos los GND (ESP-32 + fuente).
4
Sube el código: el eje debe girar una vuelta hacia un lado, pausar, y volver. Abre el Monitor Serie a 115200 para ver el sentido en cada paso.
¿Vibra o "tiembla" sin girar? Casi siempre es el orden de los cables mal puesto. Verifica que IN1=19, IN2=18, IN3=5, IN4=17 y que el constructor sea Stepper(STEPS_REV, IN1, IN3, IN2, IN4) (con IN3 e IN2 intercambiados; así lo pide este motor).

¿Gira pero se reinicia el ESP-32? Lo estás alimentando desde el 5V de la placa: pasa el + del ULN2003 a la fuente externa de 5V.
