Descripción
Demo de prueba Motor · PWM

La prueba más sencilla para tu Servo Motor SG90: hace un barrido (sweep) moviendo el brazo de 0° a 180° y de vuelta, una y otra vez. Sirve para confirmar que el servo está bien conectado, que responde a la señal y que recorre todo su rango sin trabarse. Usamos la librería ESP32Servo, que genera la señal PWM correcta para el ESP-32.

Conexiones

El SG90 trae 3 cables con colores estándar. Identifícalos por color, no por posición:

Cable del servoPin ESP-32Función
Naranja (o amarillo)GPIO 13Señal PWM
Rojo5V (VIN)Alimentación
Marrón (o negro)GNDTierra
Aliméntalo con 5V, no con 3.3V. El SG90 funciona a 5V. Conéctalo al pin 5V/VIN del ESP-32 (cuando está por USB) o al riel de 5V del Power Supply Module. La señal del cable naranja sí es de 3.3V y el servo la entiende sin problema.
Un solo servo está bien por USB, pero si conectas varios o el servo mueve algo con peso, puede pedir más corriente de la que da el ESP-32 y provocar reinicios. En ese caso aliméntalo desde una fuente externa de 5V y une las tierras (GND común).
Código Arduino
C++ / Arduino IDE (ESP-32) 
#include <ESP32Servo.h>

Servo miServo;

const int PIN_SERVO = 13;   // Cable naranja del SG90 -> GPIO 13

void setup() {
  Serial.begin(115200);

  miServo.setPeriodHertz(50);          // SG90 trabaja a 50 Hz
  miServo.attach(PIN_SERVO, 500, 2400); // Ancho de pulso min/max en microsegundos

  Serial.println("Servo SG90 listo. Iniciando barrido 0-180...");
}

void loop() {
  // De 0 a 180 grados
  for (int ang = 0; ang <= 180; ang += 2) {
    miServo.write(ang);
    Serial.printf("Angulo: %d\n", ang);
    delay(20);
  }
  delay(500);

  // De 180 de vuelta a 0
  for (int ang = 180; ang >= 0; ang -= 2) {
    miServo.write(ang);
    Serial.printf("Angulo: %d\n", ang);
    delay(20);
  }
  delay(500);
}
Qué verás en el Monitor Serie
Servo SG90 listo. Iniciando barrido 0-180... Angulo: 0 Angulo: 2 Angulo: 4 ... Angulo: 178 Angulo: 180 Angulo: 178 ...
El brazo del servo debe moverse suave y continuo de un extremo al otro, en sintonía con los ángulos del Monitor Serie. Si vibra o "zumba" en un extremo sin llegar, ajusta los valores 500 y 2400 de attach() (cada SG90 varía un poco).
Cómo usar
1
Instala la librería ESP32Servo desde el Gestor de Librerías (Programa → Incluir Librería → Administrar Bibliotecas → busca "ESP32Servo").
2
Conecta el servo por color: naranja → GPIO 13, rojo → 5V, marrón → GND.
3
Sube el código. El servo empezará a barrer de inmediato de 0° a 180° y de vuelta.
4
Abre el Monitor Serie a 115200 para ver el ángulo actual acompañando el movimiento.
¿No se mueve? Revisa que el cable rojo esté en 5V (no en 3.3V), que el GND esté conectado y que el cable de señal esté en GPIO 13. Si gira pero no llega a los extremos, juega con los microsegundos de attach().
