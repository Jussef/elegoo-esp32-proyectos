Descripción
Demo de prueba 5V Relay · Salida digital

La prueba más simple del relé: lo activa y desactiva una vez por segundo. Vas a oír el "clic" mecánico del relé y ver su LED encenderse y apagarse. Eso confirma que el módulo recibe la señal del ESP-32 y funciona, sin necesidad de conectar ningún aparato. Una vez que clickea bien, ya puedes usarlo para controlar una lámpara, una bomba, un ventilador, etc.

Conexiones
ComponentePin ESP-32Notas
Relé — IN (señal)GPIO 23Pin de control
Relé — VCC5VLa bobina del relé necesita 5V, no 3.3V
Relé — GNDGNDGND común con el ESP-32
Lado de potencia (los 3 tornillos): son COM, NO (normalmente abierto) y NC (normalmente cerrado). En este demo NO conectes nada ahí: solo queremos oír el clic. Esos bornes son para más adelante, cuando controles un aparato.
Código Arduino
C++ / Arduino IDE (ESP-32) 
#define RELE 23   // pin de senal (IN) del modulo rele

void setup() {
  pinMode(RELE, OUTPUT);
  Serial.begin(115200);
  Serial.println("Test rele: deberias oir un clic cada segundo.");
}

void loop() {
  digitalWrite(RELE, HIGH);   // cambia de estado el rele (clic)
  Serial.println("Rele -> estado A (clic)");
  delay(1000);

  digitalWrite(RELE, LOW);    // vuelve al estado anterior (clic)
  Serial.println("Rele -> estado B (clic)");
  delay(1000);
}
Cómo usar
1
Conecta solo 3 cables: IN→GPIO 23, VCC→5V y GND→GND. No conectes nada en los tornillos de potencia.
2
Sube el código. Deberías oír un "clic-clic" cada segundo y ver el LED del relé parpadear al ritmo.
3
Abre el Monitor Serie a 115200 para ver el cambio de estado A/B en cada clic.
4
¡Si clickea, tu relé funciona! Ya está listo para los proyectos de control (enchufe inteligente, riego, alarma...).
¿Active-LOW? Muchos de estos módulos se activan con LOW en vez de HIGH (el LED enciende al revés de lo que esperas). No afecta a esta prueba —igual hace clic—, pero recuérdalo en tus proyectos: si la lógica te sale invertida, cambia HIGH por LOW.
⚠️ Seguridad: este demo solo prueba el clic. No conectes 110V/220V de la pared a menos que sepas lo que haces: ese voltaje es peligroso. Para practicar el lado de potencia, usa primero algo de baja tensión (un LED con su pila, una tira LED de 12V, etc.).
