← Volver / STEPPER 03 / Seguidor de Luz
Descripción
Intermedio Motores · Sensores · Lazo cerrado

Un seguidor solar de un eje: el motor paso a paso orienta una superficie hacia la luz más intensa. Dos fotorresistores (uno a la izquierda y otro a la derecha) miden cuánta luz reciben; si uno detecta más que el otro, el motor gira hacia ese lado hasta igualar ambas lecturas. Eso es control en lazo cerrado: el sistema corrige solo hasta alcanzar el equilibrio, el mismo principio de los paneles solares que siguen al sol durante el día.

Montaje de los LDR: colócalos uno a cada lado de una pequeña pared/separador (cartón) sobre la plataforma que gira. Así, cuando la luz no viene de frente, uno queda en sombra y el otro iluminado, y el motor corrige hacia la luz.
Código Arduino
C++ / Arduino IDE (ESP-32) 
#include <Stepper.h>

const int STEPS_REV = 2048;     // 28BYJ-48: pasos por vuelta completa

#define IN1 19
#define IN2 18
#define IN3 5
#define IN4 17
#define LDR_IZQ 34              // fotorresistor izquierdo (divisor)
#define LDR_DER 35             // fotorresistor derecho (divisor)

// El orden IN1, IN3, IN2, IN4 da la secuencia correcta del 28BYJ-48 con ULN2003
Stepper motor(STEPS_REV, IN1, IN3, IN2, IN4);

const int  TOLERANCIA = 120;    // diferencia minima para moverse (zona muerta)
const int  PASO       = 16;     // cuantos pasos avanza por cada ajuste
const long LIMITE     = 1024;   // tope de giro a cada lado (~180 grados) evita enredar cables
long posActual = 0;             // posicion acumulada, en pasos

void apagarBobinas() {
  // Libera las bobinas para que el motor no se caliente entre ajustes
  digitalWrite(IN1, LOW); digitalWrite(IN2, LOW);
  digitalWrite(IN3, LOW); digitalWrite(IN4, LOW);
}

void setup() {
  Serial.begin(115200);
  motor.setSpeed(12);   // RPM
  Serial.println("Seguidor de luz listo.");
}

void loop() {
  int izq = analogRead(LDR_IZQ);
  int der = analogRead(LDR_DER);
  int dif = izq - der;          // positivo = mas luz a la izquierda

  if (abs(dif) > TOLERANCIA) {
    int dir = (dif > 0) ? -1 : 1;            // gira hacia el lado mas iluminado
    long nueva = posActual + (long)dir * PASO;

    // Solo se mueve si no supera el tope de giro a ese lado
    if (nueva >= -LIMITE && nueva <= LIMITE) {
      motor.step(dir * PASO);
      posActual = nueva;
    }
  }

  apagarBobinas();
  Serial.printf("Izq:%4d  Der:%4d  Dif:%5d  Pos:%5ld\n", izq, der, dif, posActual);
  delay(120);
}
Cómo usar
1
Arma dos divisores idénticos: cada LDR entre 3.3V y su pin (GPIO 34 / GPIO 35), y una resistencia de 10K de ese pin a GND.
2
Monta los dos LDR sobre la plataforma del motor, separados por una pared de cartón para que se hagan sombra entre sí.
3
Sube el código y abre el Monitor Serie a 115200. Mueve una linterna alrededor: el motor girará para "perseguir" la luz hasta centrarla.
4
Si la plataforma se aleja de la luz en vez de seguirla, invierte la dirección: cambia ? -1 : 1 por ? 1 : -1.
Ajustes finos: si vibra o "persigue" de más, sube TOLERANCIA (zona muerta más amplia). Si reacciona muy lento, sube PASO. El LIMITE evita que el motor dé vueltas sin fin y enrede los cables de los LDR.

Para dos ejes necesitarías un segundo motor + ULN2003 y dos LDR más (arriba/abajo), repitiendo exactamente la misma lógica de comparación.
