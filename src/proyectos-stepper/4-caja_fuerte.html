Descripción
Intermedio Motores · Teclado · OLED

Una cerradura motorizada: el motor paso a paso hace de cerrojo que gira para trabar o destrabar. Ingresas un código en el teclado matricial 4×4 y, si es correcto, el motor abre; la pantalla OLED muestra el estado y los dígitos enmascarados con asteriscos. Pulsando * borras lo escrito o cierras de nuevo, y con # confirmas el código.

Pines del teclado: el conector del keypad tiene 8 patas. De izquierda a derecha suelen ser Fila1…Fila4, Col1…Col4. Si alguna tecla responde mal, simplemente reordena los pines en los arreglos pinFilas[] / pinCols[].
Código Arduino
C++ / Arduino IDE (ESP-32) 
#include <Wire.h>
#include <Adafruit_GFX.h>
#include <Adafruit_SSD1306.h>
#include <Keypad.h>
#include <Stepper.h>

#define ANCHO 128
#define ALTO  64
Adafruit_SSD1306 oled(ANCHO, ALTO, &Wire, -1);

const int STEPS_REV = 2048;
#define IN1 19
#define IN2 18
#define IN3 5
#define IN4 17
Stepper motor(STEPS_REV, IN1, IN3, IN2, IN4);
const long PASOS_CERROJO = 1024;   // medio giro para trabar / destrabar

const byte FILAS = 4, COLS = 4;
char teclas[FILAS][COLS] = {
  {'1','2','3','A'},
  {'4','5','6','B'},
  {'7','8','9','C'},
  {'*','0','#','D'}
};
byte pinFilas[FILAS] = {13, 14, 27, 26};
byte pinCols[COLS]   = {25, 33, 32, 4};
Keypad teclado = Keypad(makeKeymap(teclas), pinFilas, pinCols, FILAS, COLS);

String CLAVE   = "1234";   // cambia tu codigo aqui
String entrada = "";
bool   trabada = true;     // arranca cerrada

void apagarBobinas() {
  digitalWrite(IN1, LOW); digitalWrite(IN2, LOW);
  digitalWrite(IN3, LOW); digitalWrite(IN4, LOW);
}

void mostrar(String l1, String l2) {
  oled.clearDisplay();
  oled.setTextColor(SSD1306_WHITE);
  oled.setTextSize(1);  oled.setCursor(0, 0);  oled.println(l1);
  oled.setTextSize(2);  oled.setCursor(0, 30); oled.println(l2);
  oled.display();
}

void destrabar() {
  if (!trabada) return;
  motor.step(PASOS_CERROJO);
  trabada = false;
  apagarBobinas();
}

void trabar() {
  if (trabada) return;
  motor.step(-PASOS_CERROJO);
  trabada = true;
  apagarBobinas();
}

void setup() {
  Serial.begin(115200);
  Wire.begin(21, 22);
  oled.begin(SSD1306_SWITCHCAPVCC, 0x3C);
  motor.setSpeed(12);
  mostrar("Caja fuerte", "CERRADA");
}

void loop() {
  char k = teclado.getKey();
  if (!k) return;

  if (k == '#') {                       // confirmar codigo
    if (entrada == CLAVE) {
      mostrar("Clave correcta", "ABIERTA");
      destrabar();
    } else {
      mostrar("Clave incorrecta", "ERROR");
    }
    entrada = "";
    delay(1500);
    if (trabada) mostrar("Ingresa clave", "----");
    else         mostrar("* para cerrar", "ABIERTA");
  }
  else if (k == '*') {                  // borrar o cerrar
    if (!trabada) {
      trabar();
      mostrar("Caja fuerte", "CERRADA");
    } else {
      entrada = "";
      mostrar("Ingresa clave", "----");
    }
  }
  else {                                // digito
    if (entrada.length() < 4) entrada += k;
    String mask = "";
    for (int i = 0; i < entrada.length(); i++) mask += "*";
    while (mask.length() < 4) mask += "-";
    mostrar("Ingresa clave", mask);
  }
}
Cómo usar
1
Instala dos librerías desde el Gestor: Adafruit SSD1306 (con su dependencia Adafruit GFX) y Keypad de Mark Stanley. Stepper.h ya viene incluida.
2
Conecta la OLED por I2C (21/22), el teclado a los 8 pines indicados y el motor a la placa ULN2003 alimentada con la fuente de 5V externa.
3
Antes de encender, deja el cerrojo en posición cerrada: el código asume que arranca trabado.
4
Escribe el código (por defecto 1234) y pulsa #. Si es correcto, el motor destraba. Pulsa * para cerrar otra vez.
5
Cambia tu clave en la línea String CLAVE = "1234"; y vuelve a subir el código.
Seguridad y mejoras: si la OLED no enciende, prueba la dirección 0x3D en oled.begin(...). Como mejora puedes bloquear el teclado unos segundos tras 3 intentos fallidos, añadir un buzzer para el "bip" de tecla y un LED rojo/verde para el estado.

Nota mecánica: el 28BYJ-48 tiene poca fuerza, así que sirve como pestillo de una cajita ligera o maqueta, no como cerradura de seguridad real.
