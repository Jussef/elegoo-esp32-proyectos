Descripción
Intermedio Motores · Sensores · Stepper

Una cortina que se abre sola al amanecer y se cierra al anochecer. El motor paso a paso enrolla y desenrolla un hilo o varilla un número fijo de vueltas. Un fotorresistor (LDR) mide la luz del ambiente y decide cuándo actuar, usando histéresis (dos umbrales) para que no se quede abriendo y cerrando sin parar al atardecer. Además, un botón permite controlarla manualmente en cualquier momento.

Código Arduino
C++ / Arduino IDE (ESP-32) 
#include <Stepper.h>

const int STEPS_REV = 2048;     // 28BYJ-48: pasos por vuelta completa

#define IN1 19
#define IN2 18
#define IN3 5
#define IN4 17
#define PIN_LDR 35              // fotorresistor (divisor de voltaje)
#define PIN_BTN 4               // boton manual (a GND)

// El orden IN1, IN3, IN2, IN4 da la secuencia correcta del 28BYJ-48 con ULN2003
Stepper motor(STEPS_REV, IN1, IN3, IN2, IN4);

const long PASOS_RECORRIDO = 3L * STEPS_REV;  // 3 vueltas para abrir/cerrar del todo
const int  UMBRAL_DIA   = 2500;   // lectura MAYOR = hay luz de dia  -> abrir
const int  UMBRAL_NOCHE = 1200;   // lectura MENOR = es de noche     -> cerrar

bool abierta = false;            // la cortina arranca CERRADA

void apagarBobinas() {
  // Libera las bobinas: evita que el motor se caliente y consuma estando parado
  digitalWrite(IN1, LOW); digitalWrite(IN2, LOW);
  digitalWrite(IN3, LOW); digitalWrite(IN4, LOW);
}

void abrir() {
  if (abierta) return;
  Serial.println("Abriendo cortina...");
  motor.step(PASOS_RECORRIDO);
  abierta = true;
  apagarBobinas();
}

void cerrar() {
  if (!abierta) return;
  Serial.println("Cerrando cortina...");
  motor.step(-PASOS_RECORRIDO);
  abierta = false;
  apagarBobinas();
}

void setup() {
  Serial.begin(115200);
  pinMode(PIN_BTN, INPUT_PULLUP);
  motor.setSpeed(12);   // RPM
  Serial.println("Cortina automatica lista (inicia CERRADA).");
}

void loop() {
  // --- Boton: control manual (invierte el estado actual) ---
  if (digitalRead(PIN_BTN) == LOW) {
    delay(30);                                   // antirrebote
    if (digitalRead(PIN_BTN) == LOW) {
      if (abierta) cerrar(); else abrir();
      while (digitalRead(PIN_BTN) == LOW) delay(10);  // espera a que sueltes
    }
  }

  // --- Modo automatico por luz, con histeresis ---
  int luz = analogRead(PIN_LDR);
  if (luz > UMBRAL_DIA && !abierta)        abrir();
  else if (luz < UMBRAL_NOCHE && abierta)  cerrar();

  Serial.printf("Luz:%4d  Cortina:%s\n", luz, abierta ? "ABIERTA" : "CERRADA");
  delay(200);
}
Cómo usar
1
La librería Stepper.h ya viene con el IDE de Arduino. No instales nada extra.
2
Arma el divisor del LDR (LDR a 3.3V, resistencia de 10K a GND, el nodo a GPIO 35) y el botón entre GPIO 4 y GND.
3
Deja la cortina físicamente cerrada antes de encender, porque el código asume que arranca cerrada (el motor no tiene sensor de posición).
4
Sube el código y abre el Monitor Serie a 115200. Tapa el LDR con la mano (simula la noche) y luego ilumínalo: la cortina cerrará y abrirá sola.
5
Pulsa el botón para abrir o cerrar manualmente en cualquier momento.
Calibra los umbrales: mira los valores de Luz: en el Monitor Serie con tu iluminación real y ajusta UMBRAL_DIA y UMBRAL_NOCHE. La separación entre ambos es la histéresis: cuanto mayor sea, menos se "duda" la cortina al atardecer.

Ajusta el recorrido: cambia PASOS_RECORRIDO (más vueltas = más tela). Si la cortina sube cuando debería bajar, invierte el signo en motor.step() o intercambia IN2 e IN3. Como mejora, añade la OLED para mostrar el estado y el nivel de luz, o un horario por NTP en vez del LDR.
