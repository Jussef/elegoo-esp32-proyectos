Descripción
Principiante Motores · ADC · Stepper

Convierte el motor paso a paso en un medidor analógico de aguja (un "gauge" como los del tablero de un auto). El motor 28BYJ-48 gira con una precisión enorme —2048 pasos por vuelta— así que puede mover una aguja a una posición exacta. Aquí la aguja sigue al potenciómetro: al girarlo de 0 a 100 %, la aguja recorre una escala de ~270°. La misma idea sirve para mostrar temperatura, nivel de señal o, sincronizando por WiFi, un reloj de un solo eje.

Alimentación del motor: El 28BYJ-48 puede pedir picos de corriente que reinician el ESP-32 si lo alimentas desde su pin 5V. Conecta el + de la placa ULN2003 a la fuente de 5V externa y une todos los GND (ESP-32 + fuente) en un punto común.
Código Arduino
C++ / Arduino IDE (ESP-32) 
#include <Stepper.h>

// 28BYJ-48: 2048 pasos por vuelta completa (modo paso completo)
const int STEPS_REV = 2048;

#define IN1 19
#define IN2 18
#define IN3 5
#define IN4 17
#define PIN_POT 34

// El orden IN1, IN3, IN2, IN4 da la secuencia correcta del 28BYJ-48 con ULN2003
Stepper motor(STEPS_REV, IN1, IN3, IN2, IN4);

const long STEPS_FONDO = 1536;  // recorrido de la aguja (~270 grados de escala)
long posActual = 0;             // posicion actual de la aguja, en pasos

void setup() {
  Serial.begin(115200);
  motor.setSpeed(12);   // velocidad en RPM (10-15 es estable para este motor)
  Serial.println("Indicador listo. Coloca la aguja en 0 antes de encender.");
}

void loop() {
  int lectura = analogRead(PIN_POT);                   // 0..4095
  long objetivo = map(lectura, 0, 4095, 0, STEPS_FONDO);
  long delta = objetivo - posActual;

  // Zona muerta: ignora micro-cambios para que la aguja no tiemble
  if (abs(delta) > 4) {
    motor.step(delta);     // mueve SOLO la diferencia hasta el objetivo
    posActual = objetivo;
    int pct = map(lectura, 0, 4095, 0, 100);
    Serial.printf("Pot:%4d  Paso:%4ld  (%d%%)\n", lectura, posActual, pct);
  }

  delay(50);
}
Cómo usar
1
La librería Stepper.h ya viene incluida con el IDE de Arduino, no necesitas instalar nada.
2
Conecta el motor 28BYJ-48 a la placa ULN2003 (cable blanco de 5 pines, entra de una sola forma). Cablea las 4 entradas IN1–IN4 al ESP-32 y alimenta el driver con la fuente de 5V externa.
3
Pega una aguja de cartón en el eje del motor. Antes de encender, gírala con la mano hasta el inicio de tu escala (posición 0). El motor no tiene sensor: asume que arranca en 0.
4
Sube el código y gira el potenciómetro lentamente: la aguja se moverá a la posición proporcional sobre la escala.
5
Abre el Monitor Serie a 115200 baudios para ver la lectura del potenciómetro, el paso actual y el porcentaje.
Tip: Si la aguja gira al revés, intercambia IN2 e IN3 en el constructor Stepper(...). ¿Quieres una escala completa de 360°? Sube STEPS_FONDO a 2048.

Conviértelo en reloj o termómetro: en vez del potenciómetro, alimenta objetivo con los segundos sincronizados por NTP (una vuelta = 60 s) o con la temperatura del DHT11 mapeada a la escala. La lógica de mover solo el delta es la misma.
