Descripción
Principiante WiFi · Redes

El ESP-32 crea su propio punto de acceso WiFi. Cuando alguien se conecta, todo el tráfico DNS se redirige automáticamente al ESP-32, que muestra una página de bienvenida — igual que los portales de hoteles o aeropuertos. No se necesita router ni internet.

Componentes
ESP-32 Board1 PC
Cable USB A → C1 PC
Conexiones

Sin conexiones externas — solo se usa el ESP-32 conectado por USB a la computadora.

Código Arduino
C++ / Arduino IDE (ESP-32) 
#include <WiFi.h>
#include <WebServer.h>
#include <DNSServer.h>

const char* AP_SSID  = "ESP32-Portal";   // Nombre del punto de acceso WiFi
const char* AP_PASS  = "";               // Sin contraseña (portal abierto)
const byte  DNS_PORT = 53;

DNSServer dnsServer;
WebServer server(80);

// Pagina HTML que se muestra al conectarse
const char PORTAL[] = R"rawliteral(
<!DOCTYPE html><html lang="es">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width,initial-scale=1">
<title>Portal ESP32</title>
<style>
  body { font-family:sans-serif; background:#111; color:#fff; display:flex;
         align-items:center; justify-content:center; min-height:100vh; margin:0; }
  .card { background:#1a1a2e; border-radius:16px; padding:32px;
          max-width:360px; width:90%; text-align:center; }
  h1 { color:#22d3ee; margin-bottom:12px; }
  p  { color:rgba(255,255,255,.65); margin-bottom:24px; font-size:14px; line-height:1.5; }
  button { background:#22d3ee; color:#111; border:none; border-radius:10px;
           padding:13px 28px; font-size:15px; font-weight:700; cursor:pointer; width:100%; }
</style>
</head>
<body>
<div class="card">
  <h1>Portal ESP32</h1>
  <p>Conectado al punto de acceso ESP32.<br>Haz clic para ingresar.</p>
  <button onclick="this.textContent='Conectado!'">Ingresar</button>
</div>
</body></html>
)rawliteral";

void setup() {
  Serial.begin(115200);

  WiFi.mode(WIFI_AP);
  WiFi.softAP(AP_SSID, AP_PASS);

  Serial.println("=== Portal WiFi Cautivo ===");
  Serial.print("SSID: ");
  Serial.println(AP_SSID);
  Serial.print("IP:   ");
  Serial.println(WiFi.softAPIP());

  // Redirige todo el DNS al ESP-32
  dnsServer.start(DNS_PORT, "*", WiFi.softAPIP());

  // Cualquier URL devuelve la pagina del portal
  server.onNotFound([]() {
    server.send(200, "text/html; charset=utf-8", PORTAL);
  });

  server.begin();
  Serial.println("Listo. Conectate al WiFi y abre cualquier URL.");
}

void loop() {
  dnsServer.processNextRequest();
  server.handleClient();
}
Cómo usar
1
Instala las librerías WiFi, WebServer y DNSServer (incluidas en el paquete ESP32 de Arduino IDE).
2
Cambia AP_SSID por el nombre de red WiFi que quieras mostrar (ej: "Gratis-Cafe").
3
Sube el código al ESP-32. Verifica en el Monitor Serie (115200 baud) que aparezca la IP del punto de acceso.
4
Desde tu teléfono o computadora, busca la red WiFi con el nombre que pusiste, conéctate y abre cualquier URL — el portal aparecerá automáticamente.
Tip: Puedes personalizar la página HTML dentro de la variable PORTAL[]. Agrega un formulario, un logo o cualquier contenido que quieras mostrar a los visitantes.
