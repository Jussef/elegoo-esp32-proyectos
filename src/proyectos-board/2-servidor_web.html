Descripción
Principiante WiFi · IoT

El ESP-32 se conecta a tu red WiFi y levanta un servidor web en su IP local. Desde cualquier navegador en la misma red puedes encender y apagar el LED integrado en tiempo real. La página se actualiza automáticamente sin recargar.

Componentes
ESP-32 Board1 PC
LED (cualquier color)1 PC
Resistor 330 Ω1 PC
Protoboard + cables—
Conexiones
ComponentePin ESP-32Notas
LED — ánodo (+) GPIO 2 En serie con la resistencia de 330 Ω
Resistor 330 Ω GND Entre el cátodo del LED y GND
LED — cátodo (−) GND (vía resistor) La pata más corta del LED
Alternativa: El ESP-32 tiene un LED integrado en GPIO 2. Si no quieres armar el circuito en el protoboard, simplemente observa el LED de la placa.
Código Arduino
C++ / Arduino IDE (ESP-32) 
#include <WiFi.h>
#include <WebServer.h>

const char* SSID     = "TU_WIFI";     // Nombre de tu red WiFi
const char* PASSWORD = "TU_CLAVE";    // Contrasena de tu red WiFi
const int   LED_PIN  = 2;             // LED integrado del ESP-32

WebServer server(80);
bool ledOn = false;

// Pagina web de control
const char PAGE[] = R"rawliteral(

void setup() {
  Serial.begin(115200);
  pinMode(LED_PIN, OUTPUT);

  WiFi.begin(SSID, PASSWORD);
  Serial.print("Conectando a WiFi");
  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }
  Serial.println("\nConectado!");
  Serial.print("Abre en tu navegador: http://");
  Serial.println(WiFi.localIP());

  server.on("/", []() {
    server.send(200, "text/html; charset=utf-8", PAGE);
  });
  server.on("/toggle", []() {
    ledOn = !ledOn;
    digitalWrite(LED_PIN, ledOn);
    server.send(200, "text/plain", "ok");
  });
  server.on("/estado", []() {
    server.send(200, "application/json",
      ledOn ? "{\"on\":true}" : "{\"on\":false}");
  });
  server.begin();
}

void loop() {
  server.handleClient();
}
Cómo usar
1
Reemplaza TU_WIFI y TU_CLAVE con el nombre y contraseña de tu red WiFi.
2
Sube el código. Abre el Monitor Serie (115200 baud) y espera a que aparezca la dirección IP del ESP-32.
3
En cualquier dispositivo conectado a la misma red WiFi, abre un navegador y escribe esa IP (ej: http://192.168.1.45).
4
Presiona el botón en la página web para encender y apagar el LED del ESP-32 en tiempo real.
Tip: Puedes agregar más rutas con server.on("/ruta", handler) para controlar varios pines o mostrar datos de sensores.
