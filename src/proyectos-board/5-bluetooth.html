Descripción
Intermedio Bluetooth · Inalámbrico

El ESP-32 se empareja con tu teléfono vía Bluetooth Classic y crea un canal de comunicación serial inalámbrico. Desde una app de terminal en el celular puedes enviar comandos para controlar el LED integrado, y el ESP-32 responde con mensajes de confirmación. Todo sin cables ni WiFi.

Componentes
ESP-32 Board1 PC
Cable USB A → C (para programar)1 PC
Teléfono Android con Bluetooth—
Conexiones

Sin conexiones externas — el LED integrado en GPIO 2 de la placa ESP-32 se usa directamente.

Comandos disponibles (desde el celular)
L1
Enciende el LED integrado del ESP-32
L0
Apaga el LED integrado del ESP-32
ST
Consulta el estado actual del LED (encendido/apagado)
cualquier texto
El ESP-32 hace eco del mensaje y lo muestra en el Monitor Serie
Código Arduino
C++ / Arduino IDE (ESP-32) 
#include "BluetoothSerial.h"

// Bluetooth Classic — solo funciona en ESP-32 con soporte BT
// (no en ESP32-S2 o ESP32-C3)
#if !defined(CONFIG_BT_ENABLED) || !defined(CONFIG_BLUEDROID_ENABLED)
  #error Bluetooth no habilitado. Ve a: Arduino IDE -> Herramientas -> Partition Scheme -> Huge APP
#endif

BluetoothSerial BT;
const String DEVICE_NAME = "ESP32-BT";   // Nombre que aparece en el telefono
const int    LED_PIN     = 2;            // LED integrado del ESP-32

void procesarComando(String cmd) {
  cmd.trim();

  if (cmd == "L1") {
    digitalWrite(LED_PIN, HIGH);
    BT.println("LED: ENCENDIDO");
    Serial.println("[BT] LED encendido");

  } else if (cmd == "L0") {
    digitalWrite(LED_PIN, LOW);
    BT.println("LED: APAGADO");
    Serial.println("[BT] LED apagado");

  } else if (cmd == "ST") {
    bool estado = digitalRead(LED_PIN);
    BT.println("Estado LED: " + String(estado ? "ENCENDIDO" : "APAGADO"));

  } else if (cmd.length() > 0) {
    BT.println("Eco: " + cmd);
    Serial.println("[BT] mensaje: " + cmd);
  }
}

void setup() {
  Serial.begin(115200);
  pinMode(LED_PIN, OUTPUT);

  BT.begin(DEVICE_NAME);

  Serial.println("=== Bluetooth Serial ===");
  Serial.println("Dispositivo: " + DEVICE_NAME);
  Serial.println("Busca este nombre en tu telefono y empareja.");
  Serial.println("Comandos: L1 / L0 / ST");
  Serial.println("========================");
}

void loop() {
  // Recibe datos del telefono via Bluetooth
  if (BT.available()) {
    String msg = BT.readStringUntil('\n');
    procesarComando(msg);
  }

  // Reenvía lo que escribas en Monitor Serie al telefono
  if (Serial.available()) {
    String msg = Serial.readStringUntil('\n');
    BT.println(msg);
  }

  delay(20);
}
Cómo usar
1
Sube el código al ESP-32 y abre el Monitor Serie a 115200 baud para ver los mensajes de estado.
2
En tu teléfono Android, instala la app Serial Bluetooth Terminal (disponible en Play Store, gratis).
3
Ve a Configuración → Bluetooth de tu teléfono y empareja con el dispositivo llamado ESP32-BT. PIN: sin PIN o 1234.
4
Abre la app Serial Bluetooth Terminal, conéctate a ESP32-BT y envía comandos como L1, L0, ST.
Nota: Esta librería usa Bluetooth Classic (no BLE). Solo funciona con Android. Para iOS necesitarías usar BLE con la librería NimBLE-Arduino.
