Descripción
Intermedio WiFi · OTA · DevOps

OTA (Over-The-Air) permite actualizar el firmware del ESP-32 por WiFi desde el Arduino IDE, sin necesidad de conectar el cable USB cada vez. Ideal cuando el ESP-32 está montado en un lugar difícil de alcanzar. El LED integrado parpadea para indicar que el sistema está activo y listo para recibir actualizaciones.

Componentes
ESP-32 Board1 PC
Cable USB A → C (solo para la primera carga)1 PC
Red WiFi con acceso a internet—
Conexiones

Sin conexiones externas — el LED integrado en GPIO 2 parpadea para confirmar que el sistema está activo.

Flujo de trabajo OTA
Primera vez
Sube el código por USB con el cable
›
El ESP-32 se conecta
Al WiFi y anuncia su presencia en la red
›
Futuras updates
Arduino IDE → Puerto (red) → Subir sin USB
Código Arduino
C++ / Arduino IDE (ESP-32) 
#include <WiFi.h>
#include <ArduinoOTA.h>

const char* SSID         = "TU_WIFI";     // Nombre de tu red WiFi
const char* PASSWORD     = "TU_CLAVE";    // Contrasena de tu red WiFi
const char* OTA_HOSTNAME = "esp32-ota";   // Nombre en la lista de puertos de Arduino IDE
const char* OTA_PASSWORD = "1234";        // Contrasena para autorizar la actualizacion

const int LED_PIN = 2;   // LED integrado del ESP-32

void setupOTA() {
  ArduinoOTA.setHostname(OTA_HOSTNAME);
  ArduinoOTA.setPassword(OTA_PASSWORD);

  ArduinoOTA.onStart([]() {
    String tipo = (ArduinoOTA.getCommand() == U_FLASH) ? "firmware" : "filesystem";
    Serial.println("\nIniciando OTA (" + tipo + ")...");
    digitalWrite(LED_PIN, LOW);   // Apaga LED durante la actualizacion
  });

  ArduinoOTA.onEnd([]() {
    Serial.println("\nActualizacion completada! Reiniciando...");
  });

  ArduinoOTA.onProgress([](unsigned int progress, unsigned int total) {
    int pct = progress / (total / 100);
    Serial.printf("\rProgreso: %3d%%  [", pct);
    for (int i = 0; i < 20; i++) Serial.print(i < pct / 5 ? '#' : '.');
    Serial.print("]");
  });

  ArduinoOTA.onError([](ota_error_t error) {
    Serial.println("\nError OTA:");
    if      (error == OTA_AUTH_ERROR)    Serial.println("  Contrasena incorrecta");
    else if (error == OTA_BEGIN_ERROR)   Serial.println("  Error al iniciar");
    else if (error == OTA_CONNECT_ERROR) Serial.println("  Error de conexion");
    else if (error == OTA_RECEIVE_ERROR) Serial.println("  Error al recibir");
    else if (error == OTA_END_ERROR)     Serial.println("  Error al finalizar");
  });

  ArduinoOTA.begin();
}

void setup() {
  Serial.begin(115200);
  pinMode(LED_PIN, OUTPUT);

  WiFi.begin(SSID, PASSWORD);
  Serial.print("Conectando a WiFi");
  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }
  Serial.println("\nConectado!");

  setupOTA();

  Serial.println("=== OTA listo ===");
  Serial.print("IP:       "); Serial.println(WiFi.localIP());
  Serial.print("Hostname: "); Serial.println(OTA_HOSTNAME);
  Serial.println("En Arduino IDE -> Herramientas -> Puerto");
  Serial.println("Selecciona: " + String(OTA_HOSTNAME) + " (network)");
  Serial.println("=================");
}

void loop() {
  ArduinoOTA.handle();   // SIEMPRE al inicio del loop para recibir updates

  // Parpadeo del LED — indica que el sistema esta activo
  static unsigned long ultimo = 0;
  static bool estado = false;
  if (millis() - ultimo > 1000) {
    ultimo = millis();
    estado = !estado;
    digitalWrite(LED_PIN, estado);
  }
}
Cómo usar
1
Cambia TU_WIFI y TU_CLAVE con tu red WiFi. Puedes también cambiar OTA_PASSWORD.
2
Sube el código por USB la primera vez. El Monitor Serie mostrará la IP y el nombre de red del ESP-32.
3
Para subir código por WiFi: en Arduino IDE ve a Herramientas → Puerto y selecciona esp32-ota (network) en lugar del puerto COM.
4
Presiona Subir normalmente. Se te pedirá la contraseña OTA (la que pusiste en OTA_PASSWORD).
5
El LED se apagará durante la actualización y el Monitor Serie mostrará la barra de progreso. Al terminar, el ESP-32 se reinicia.
Importante: Tu computadora y el ESP-32 deben estar en la misma red WiFi para que OTA funcione. Asegúrate también de dejar ArduinoOTA.handle() al inicio de cada loop() en tus proyectos futuros.
