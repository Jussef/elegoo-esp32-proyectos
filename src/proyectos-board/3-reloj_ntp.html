Descripción
Principiante WiFi · Internet

El ESP-32 se conecta a internet y sincroniza la hora exacta con servidores NTP (Network Time Protocol). No necesita ningún módulo de reloj (RTC) — el tiempo se obtiene automáticamente y se muestra cada segundo en el Monitor Serie con fecha y hora completas.

Componentes
ESP-32 Board1 PC
Cable USB A → C1 PC
Conexión a internet (WiFi)—
Conexiones

Sin conexiones externas — solo el ESP-32 conectado por USB. La salida se ve en el Monitor Serie.

Zona horaria — ajusta según tu país
México Centro
-6 * 3600
Colombia / Perú / Ecuador
-5 * 3600
Venezuela
-4 * 3600
Argentina / Chile
-3 * 3600
España (invierno)
+1 * 3600
España (verano DST)
DST = 3600
Código Arduino
C++ / Arduino IDE (ESP-32) 
#include <WiFi.h>
#include <time.h>

const char* SSID     = "TU_WIFI";    // Nombre de tu red WiFi
const char* PASSWORD = "TU_CLAVE";   // Contrasena de tu red WiFi

// Ajusta el offset GMT segun tu pais (ver tabla en la pagina)
// Colombia / Peru / Ecuador = -5 * 3600
const long  GMT_OFFSET_SEC      = -5 * 3600;
const int   DAYLIGHT_OFFSET_SEC = 0;       // 3600 si tu pais usa horario de verano
const char* NTP_SERVER          = "pool.ntp.org";

const char* DIAS[]  = { "Dom","Lun","Mar","Mie","Jue","Vie","Sab" };
const char* MESES[] = { "Ene","Feb","Mar","Abr","May","Jun",
                         "Jul","Ago","Sep","Oct","Nov","Dic" };

void printTime() {
  struct tm t;
  if (!getLocalTime(&t)) {
    Serial.println("Error: no se pudo obtener la hora");
    return;
  }
  Serial.printf("%s %02d %s %04d  -  %02d:%02d:%02d\n",
    DIAS[t.tm_wday],
    t.tm_mday,
    MESES[t.tm_mon],
    t.tm_year + 1900,
    t.tm_hour,
    t.tm_min,
    t.tm_sec
  );
}

void setup() {
  Serial.begin(115200);

  WiFi.begin(SSID, PASSWORD);
  Serial.print("Conectando a WiFi");
  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }
  Serial.println("\nConectado! Sincronizando hora NTP...");

  configTime(GMT_OFFSET_SEC, DAYLIGHT_OFFSET_SEC, NTP_SERVER);

  // Espera la primera sincronizacion (max 10 intentos)
  struct tm t;
  int tries = 0;
  while (!getLocalTime(&t) && tries++ < 10) {
    Serial.print(".");
    delay(1000);
  }

  if (getLocalTime(&t)) {
    Serial.println("\nHora sincronizada!\n");
  } else {
    Serial.println("\nTimeout NTP - verifica la conexion.\n");
  }
}

void loop() {
  printTime();
  delay(1000);
}
Cómo usar
1
Cambia TU_WIFI y TU_CLAVE con tu red y contraseña WiFi.
2
Ajusta GMT_OFFSET_SEC según tu país usando la tabla de arriba (ej: Colombia = -5 * 3600).
3
Sube el código y abre el Monitor Serie a 115200 baud. Espera unos segundos a que se sincronice.
4
Verás la hora actualizada cada segundo: Lun 16 May 2026 - 14:32:07
Tip: Puedes usar este proyecto como base para un reloj OLED — combínalo con el proyecto de la pantalla OLED para mostrar la hora en hardware en lugar del Monitor Serie.
